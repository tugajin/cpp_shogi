# cpp_shogi
以下のプログラムを参考にしています。
- やねうら王
- Appery
- senpai 2.0
- dlshogi
- Bonanza6.0

---

以下の環境でコンパイルしています。
- cmake version 3.10以上
 - Pytorch 1.4

- Linuxの場合
  - gcc version 7.5.0
- Windowsの場合
  - Visual Studio 2019

---
コンパイル方法
1. srcディレクトリに移動する
```
cd src
```
2. buildディレクトリを作成する
```
make build
cd build
```
3. cmakeする
```
cmake ..
```
4. makeする
```
make
```

5.動かす
```
./cpp_shogi
```
